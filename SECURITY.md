# Security Policy

## Supported versions

MaintainerForge is currently pre-1.0. Security fixes target the latest main branch and the latest tagged release.

## Reporting a vulnerability

Please open a private security advisory on GitHub or contact the maintainer by email. Do not publish exploit details before a fix is available.

## Scope

MaintainerForge is intended for repositories the maintainer owns or is authorized to review. It must not be used to scan, probe, or analyze third-party repositories without permission.
